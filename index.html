LU02

Zusammenfassung

In LU02 haben wir die Struktur vom HTML Code gelernt, also articles, sections etc.

Codebeispiel

    <!DOCTYPE html>
    <html lang="de">
      <head>
        <meta charset="UTF-8">
        <title>LU02.A01 - Bereiche einsetzen</title>
        <link rel="stylesheet" href="bereiche.css">
      </head>
      <body>
        <header>
          Ich bin die Kopfzeile der Seite
          <nav>
            Hier steht die Navigation
          </nav>
        </header>

        <main>
          Der Hauptinhalt folgt hier.
          <article>
            Ich bin der erste Artikel.
            <section>
              Als Abschnitt gehöre ich zum ersten Artikel.
            </section>
            <section>
              Auch ich bin ein Abschnitt.
            </section>
          </article>
          <article>
            Der zweite Artikel bin ich.
            <section>
              Abschnitt eins im Artikel zwei.
            </section>
          </article>
        </main>

        <footer>
          Die Fusszeile schliesst alles ab.
        </footer>
      </body>
    </html>

Zusammenfassung

In LU03 haben wir uns den Unterschied zwischen Block und Inline Elementen angeschaut, sowie einige kennengelernt.

Codebeispiel

                <!DOCTYPE html>
    <html lang='de'>
      <head>
        <meta charset='UTF-8'>
        <title>LU03.A01 - Portfolio</title>
      </head>
      <body>
      <h1>Mein Portfolio</h1>

      <section>
        <h2>Über mich</h2>
        <img src="me.png" alt="Ein Bild von mir" width="200" height="200">
        <p>Ich bin ein professioneller Entwickler, spezialisiert auf das <strong>Durchsuchen</strong> von Code anderer Entwickler nach <strong>Fehlern</strong> und <strong>Sicherheitslücken</strong>. Ich liebe es, meine Fähigkeiten zu nutzen, um Software sicherer und effizienter zu machen.</p>
      </section>

      <section>
        <h2>Meine Hobbys</h2>
        <ul>
          <li>Programmieren</li>
          <li>Musik hören</li>
          <li>Wandern</li>
        </ul>
      </section>

      <section>
        <h2>Lieblingsessen</h2>
        <ol>
          <li>Pizza</li>
          <li>Sushi</li>

      </body>
