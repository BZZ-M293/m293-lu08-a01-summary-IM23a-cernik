LU01

Zusammenfassung

In der LU01 haben wir die Einstellungen von Web Storm, sowie gelernt wie man Hello World mit HTML anzeigen lässt.

Codebeispiel

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Hello</title>
</head>
<body>
    Hello World
</body>
</html>

LU02

Zusammenfassung

In LU02 haben wir die Struktur vom HTML Code gelernt, also articles, sections etc.

Codebeispiel

    <!DOCTYPE html>
    <html lang="de">
      <head>
        <meta charset="UTF-8">
        <title>LU02.A01 - Bereiche einsetzen</title>
        <link rel="stylesheet" href="bereiche.css">
      </head>
      <body>
        <header>
          Ich bin die Kopfzeile der Seite
          <nav>
            Hier steht die Navigation
          </nav>
        </header>

        <main>
          Der Hauptinhalt folgt hier.
          <article>
            Ich bin der erste Artikel.
            <section>
              Als Abschnitt gehöre ich zum ersten Artikel.
            </section>
            <section>
              Auch ich bin ein Abschnitt.
            </section>
          </article>
          <article>
            Der zweite Artikel bin ich.
            <section>
              Abschnitt eins im Artikel zwei.
            </section>
          </article>
        </main>

        <footer>
          Die Fusszeile schliesst alles ab.
        </footer>
      </body>
    </html>

LU04

Zusammenfassung

In LU04 haben wir uns mit dem Codingstyle für HTML beschäftigt. Dort haben wir wichtige befehle wie ctrl + alt + l kennengelernt.

Codebeispiel

    <!DOCTYPE html>
    <html lang="de">
      <head>
        <meta charset="UTF-8">
        <title>Gleitschirmfliegen - Tipps und Ausrüstung</title>
      </head>
      <body>
        <h1>Gleitschirmfliegen - Einsteigertipps und Ausrüstung</h1>
        <h2>Einführung ins Gleitschirmfliegen</h2>
        <p>Gleitschirmfliegen ist ein beliebter Luftsport, bei dem der Pilot mit einem Gleitschirm durch die Lüfte gleitet. Es erfordert Übung und eine spezielle Ausrüstung, um sicher fliegen zu können.</p>
        <h2>Ausrüstung</h2>
        <p>Um Gleitschirmfliegen zu können, benötigen Sie eine Grundausstattung. Diese umfasst in der Regel einen Gleitschirm, einen Gurtzeug und einen Helm. Zusätzlich empfiehlt sich der Kauf eines Notschirms, einer Funkanlage und eines GPS-Geräts.</p>
        <h2>Tipps für Anfänger</h2>
        <ol>
          <li>Wählen Sie einen geeigneten Startplatz</li>
          <li>Üben Sie das Starten und Landen</li>
          <li>Lernen Sie die Wetterbedingungen kennen</li>
        </ol>
        <h3>Beliebte Fluggebiete</h3>
        <ul>
          <li>Österreich</li>
          <li>Schweiz</li>
          <li>Deutschland</li>
        </ul>
      </body>
    </html>

Lu07

Zusammenfassung

In LU07 haben wir uns erweitere Formularelemente wie z.B. select und datalist angeschaut.

Codebeispiel

    <!DOCTYPE html>
    <html lang='de'>
    <head>
      <meta charset='UTF-8'>
      <title>Hotelzimmer-Buchungssystem</title>

    </head>
    <body>

    <h1>Hotelzimmer-Buchungssystem</h1>

    <form action='https://it.bzz.ch/demo/formdata/index.php' method='post'>
      <!-- Zimmertypen -->
      <fieldset>
        <legend>Zimmertyp</legend>
        <select id='zimmertyp' name='Zimmertyp[]' multiple required>
          <option value='einzelzimmer'>Einzelzimmer</option>
          <option value='doppelzimmer'>Doppelzimmer</option>
          <option value='suite'>Suite</option>
        </select>
      </fieldset>

      <!-- Zusatzleistungen -->
      <fieldset>
        <legend>Zusatzleistungen</legend>
        <input type='checkbox' id='fruehstueck' name='zusatzleistungen[]' value='fruehstueck'>
        <label for='fruehstueck'>Frühstück</label><br>
        <input type='checkbox' id='transfer' name='zusatzleistungen[]' value='transfer'>
        <label for='transfer'>Flughafentransfer</label><br>
        <input type='checkbox' id='spa' name='zusatzleistungen[]' value='spa'>
        <label for='spa'>Spa-Zugang</label>
      </fieldset>

      <!-- Submit Button -->
      <input type='submit' value='Buchen'>
    </form>


    </body>
    </html>
