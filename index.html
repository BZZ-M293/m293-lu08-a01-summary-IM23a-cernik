Zusammenfassung

In der LU01 haben wir die Einstellungen von Web Storm, sowie gelernt wie man Hello World mit HTML anzeigen lässt.

Codebeispiel

<!DOCTYPE html>
<html lang="en">

</body>
</html>

LU02

Zusammenfassung

In LU02 haben wir die Struktur vom HTML Code gelernt, also articles, sections etc.

Codebeispiel

    <!DOCTYPE html>
    <html lang="de">
      <head>
        <meta charset="UTF-8">
        <title>LU02.A01 - Bereiche einsetzen</title>
        <link rel="stylesheet" href="bereiche.css">
      </head>
      <body>
        <header>
          Ich bin die Kopfzeile der Seite
          <nav>
            Hier steht die Navigation
          </nav>
        </header>

        <main>
          Der Hauptinhalt folgt hier.
          <article>
            Ich bin der erste Artikel.
            <section>
              Als Abschnitt gehöre ich zum ersten Artikel.
            </section>
            <section>
              Auch ich bin ein Abschnitt.
            </section>
          </article>
          <article>
            Der zweite Artikel bin ich.
            <section>
              Abschnitt eins im Artikel zwei.
            </section>
          </article>
        </main>

        <footer>
          Die Fusszeile schliesst alles ab.
        </footer>
      </body>
    </html>
